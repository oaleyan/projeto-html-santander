Sobre Este Documento

Este é um parágrafo que demonstra o uso de algumas tags de texto. Aqui, podemos ter um texto em destaque, uma parte com letras menores, e uma palavra em itálico. É importante notar que este conteúdo é sublinhado e algumas informações são muito importantes.

Também podemos usar a tag para riscar um texto que foi removido ou está incorreto.
